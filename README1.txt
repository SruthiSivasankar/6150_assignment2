This is a website for professional portfolio with 3 pages- home,about and portfolio
<nav> is used to navigitate between pages and to access contact class
<div> and <nav> tags are used to create the column layout
Float and overflow properties are used for the column class
Fixed positioning is used for navigation elements and location sticker in home page
Relative positioning is used for certificates in the gallery
<a> is used to add tel and mailto options in the contact
media queries are used for column class to make website responsive for devices
Hover property is used for every column, buttons and sections within the website
Stickers and icons are created for hyperlinks to make the page interactive
background-image property is used to distinguish between columns
